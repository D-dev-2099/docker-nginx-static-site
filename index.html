Welcome to My Static Docker Website

This is a simple static website served using Nginx in a Docker container.

This project demonstrates how to containerize and deploy a simple static website using Docker and Nginx. A basic HTML page (index.html) is served through an Nginx server running inside a lightweight Docker container. The Dockerfile defines the image build process, which uses the official nginx:alpine base image and copies the static content into the default web directory. This setup is ideal for understanding the basics of Docker image creation, container lifecycle, and how web servers like Nginx can be used to serve static content efficiently. It forms a foundational DevOps project that introduces containerization and web hosting in a practical, hands-on way.
